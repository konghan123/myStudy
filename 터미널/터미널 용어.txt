터미널 용어
ls 폴더리스트 / ls - name 폴더 이름만 / ls 폴더이름  해당 폴더안을 볼 수잇음 / ls -force 숨겨진 파일
cd (폴더) 해당 폴더로 이동  = cd .\ 안 폴더로 이동
cd ..(\ )밖 폴더로 이동
cd ~ 최상위 폴더로 이동
cd - 이전 폴더로 이동
man 메뉴
q 원상태로 돌리기
clear 스크린 청소
pwd 현재 나의 경로
explorer . 현재 위치를 파일로 열기
man ls 겟차일드
mkdir 새폴더(경로) 만들기 / mkdir (새폴더)/(새폴더)/(새폴더) 연속으로 만들 수 있다.
get-childitem -file -filter "찾고싶은 것(ex. *.txt)" -recurse  찾고 싶은 문서가 있을때
get-command (파일) = gcm (파일) 해당 파일의 실행경로를 알 수 있다.
new-item (만들고 싶은 파일) 새로운 파일을 만들 수있슴 
cat 파일이름 파일 안에 있는 내용들을 확인 가능
echo "문장" > (파일이름) 파일안에 문장을 덮어 씌운다
echo "문장" >> (파일이름) 파일안에 문장을 추가한다.
cp (파일) (폴더) 파일을 해당폴더에 복사
mv (파일) (폴더) 파일을 해당 폴더에 이동
rm 파일(폴더)을 삭제
select-string 찾고싶은 것(ex. *.txt) -pattern "원하는 문자" 파일에서 원하는 문자를 찾을 수 있슴
$env:(원하는 변수명) = "원하는 값"   환경 변수 설정
ls env:  설정된 모든 환경 변수를 볼 수 있다
cd $env:(변수명) 환경변수로 이동
$env:(변수명) = "" 환경변수 삭제
